### Cloudflare 란?

이걸 읽는분이 Cloudflare를 모를리가 없지만 간단하게 정리하자면<br/>

- **도메인 등록, DNS 관리**
- **WAF & DDoS 방어**
- **콘텐츠 전송 네트워크 (CDN)**
- **SSL/TLS 암호화**
- **캐싱, 속도 및 트래픽 자동 최적화**
- **배포**

웹 서비스에서 필요한 대부분의 서비스를 제공하는 플랫폼입니다.<br/>
다만, 저는 Cloudflare 에서 배포를 제외한 기능을 활용했습니다.

### Cloudflare를 선택한 이유

1. **도메인 가격 정책**
    - 도메인 등록 비용이 ** 추가 요금 없이 원가 그대로 제공됨. noteapp.org는 10달러쯤에 구매함.**.
    - 일반적인 도메인 등록업체와 달리, 갱신 시 추가 비용이 붙지 않고 처음 구매한 가격 그대로 유지됨.

<br/>

2. **도메인 구매 및 관리가 한곳에서 이뤄짐**
    - 따라서 별도의 네임서버 변경 없이 등록, DNS 설정, 보안 설정 적용이 **즉시** 적용되고  사용이 가능해 매우 간편함.

<br/>

3. **쉬운 설정 및 무료 기능**
    - SSL, DNSSEC, WAF, 봇차단, HTTP 헤더 무결성 검사, 속도 제한 규칙(Too Many Request와 같은) 등 주요 기능이 기본 제공되며, 대시보드 덕분에 설정이 매우 직관적임.
    - 즉, 저와 같은 초보자가 웹 환경에 서버를 노출시킬때 안전성을 확보하기에 가장 적합하다고 생각되었습니다.
