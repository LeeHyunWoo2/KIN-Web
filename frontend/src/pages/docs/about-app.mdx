# 앱 소개



## ❓ 어떤 앱인가요?

Keep Idea Note(이하 KIN)는 개인적 필요에 의해 개발하게 된 노트 관리 웹 애플리케이션입니다.<br/>
검색어, 카테고리 및 태그 등 강력한 필터링을 제공하여 한번 기록한 내용을 쉽고 빠르게 찾아볼 수 있습니다.

📌 **특징 요약**
1. **카테고리 + 태그 필터링 시스템** → 자료 정리 & 검색 최적화
2. **메모 & 에디터 모드** → 간단한 메모 모드, 디테일한 서식을 위한 에디터 모드 제공
3. **데이터 백업 & 보안 강화** → 데이터 보호 & 장기 보관 가능

<br/>

##  📝 노트 앱으로 결정한 배경

교육을 받으며 프로그래밍이 적성과 취향에 너무 잘 맞아 푹 빠져서 학습을 진행하였습니다. <br/>
그 결과, 짧은 기간 동안 많은 양의 정보를 습득하게 되었지만, 그 대가로 쌓인 다양한 유형의 자료들이 아무리 정리를 해놔도 쉽게 못 찾는 상황에 이르렀습니다.
특히 배운 지 오래된 정보를 찾아내야 할 때 정말 불편했습니다.<br/>

기존의 노트 앱들은 검색 기능과 분류 시스템이 아쉬웠고, 정보를 찾기 위해 헤매는 상황이 생겼습니다.<br/>
KIN은 이러한 문제를 해결하기 위해 **"검색과 정리"** 에 초점을 맞춘 노트 앱입니다.

<br/>

## 🛠️ 핵심 기능

### 1️⃣ 필터링 시스템

- 다중 태그 필터링 지원 (ex. `Java` + `알고리즘` 태그 검색 가능)
- **태그 + 키워드 조합 검색** 가능
- **카테고리 분류**를 통해 문서 정리 최적화
- **즐겨찾기** 및 날짜순 등 추가적인 정렬 기능 제공

**[기능 Gif 시연](https://noteapp.org/docs/notes/filter)**

---

### 2️⃣ 다양한 편집 기능

- 기본 메모 모드 (빠른 기록용)
- **Rich Text Editor 지원** (Plate Editor 기반)
- 코드 블록, 인용문, 체크리스트 등 다양한 서식 제공

---

### 3️⃣ 데이터 안전성

- 자동 백업 기능 (클라우드 저장)
- 데이터 유실 방지 (수동 백업 지원)
- 개인 정보 보호 (외부 접근 차단)

<br/>

## ❓ 어떻게 활용할 수 있나요?

이 앱은 **개인 학습, 업무 기록, 아이디어 정리** 등 다양한 용도로 활용할 수 있습니다.
특히, 평소에 메모를 자주 하여 정보가 흩뿌려져 있는 분에게 좋습니다.

✔️ **공부한 내용을 정리할 때**<br/>
✔️ **개발자가 작성해둔 코드 스니펫을 빠르게 찾을 때**<br/>
✔️ **중요한 아이디어를 빠르게 기록할 때**<br/>

<br/>

## ♾️ 앞으로의 방향

KIN은 개발 중단 없이 꾸준히 업데이트할 것입니다. (취업하더라도)

앞으로 버전 관리 기능, PC 버전 (Electron), 모바일 버전, OTP도입, 더 가벼운 에디터로 변경 등<br/>
더욱 강력한 노트 관리 환경을 제공할 계획입니다.
