# 개요 - 백엔드

백엔드 개발 과정에서의 선택과 세부 사항을 정리한 문서입니다.

<br/>

### 백엔드 기술로 분류된 프론트엔드 작업

<Callout type="info" emoji="ℹ️">
  일부 로직은 클라이언트 및 프론트엔드 서버에서 실행되지만, 여러 이유로 **백엔드 기술로 분류**했습니다.
</Callout>

<br/>

제 프로젝트에서는 백엔드의 부담을 덜기 위해 특정 데이터 가공 및 분류를 클라이언트에서 담당하고,<br/>
프론트엔드 서버에서 담당하는 통신, 헤더 설정, 프록싱 등의 작업이 있는 등.

백엔드 기술과 동일한 원리를 적용한 프론트엔드 로직이 많습니다.

이 문서의 목적은 로직 설계 및 구현 과정에 대한 이해도를 강조하는 것입니다.<br/>

따라서 이런 요소를 기술적으로 분류하자면 프론트엔드가 맞지만, 이 문서의 의도를 감안하여 백엔드로 분류하였습니다.<br/>
