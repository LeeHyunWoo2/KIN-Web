**노트 기능**
  - 노트 CRUD, 자동 저장, 필터링, 태그, 검색 등 설명
  - Plate.js 기반 에디터 구현 방식, 기본 텍스트 방식
  - 바이너리 데이터 압축 방식 (`noteCompressor.js` 사용)
  - PouchDB 활용한 클라이언트 로컬 데이터 저장
  - 클라우드 동기화 및 서버와의 데이터 싱크 로직
  - 등 여러가지 기능들의 소개를 업로드 할 예정입니다.
